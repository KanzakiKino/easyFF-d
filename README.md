easyFF-d
----

easyFF-d is D bindings of a wrapper library for FFmpeg.  
Original written in C is [here](https://github.com/KanzakiKino/easyFF).

**easyFF is now developing.**

## Requirements
- libavcodec
- libavformat
- libavutil
- libeasyff

## Using with DUB

Write this in `dependencies` section.

    "easyFF-d": "~>0.0.1"
